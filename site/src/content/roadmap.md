Esta página lista o que está no horizonte do Toolbox. As ideias estão organizadas por área — nenhuma tem data definida ainda. Se algo aqui te interessa muito, abra uma issue no GitHub para a gente priorizar.

## Conforto no dia a dia

- **Atalho configurável** — escolher o próprio atalho (hoje é fixo em `Ctrl+Space`)
- **Buscar com erro de digitação** — digitar "gut" e aparecer "git" (busca fuzzy)
- **Arrastar para reordenar** — reorganizar comandos com o mouse
- **Pastas/grupos** — agrupar comandos por tema (trabalho, estudo, pessoal)
- **Aliases** — `g` para `google`, `gh` para `github` (parcialmente já funciona)
- **Modo barra de tarefas** — deixar só a barra de busca fixa na tela

## Plugins mais poderosos

- **Hot-reload** — instalar plugin sem reiniciar o Toolbox
- **Comunicação de volta** — plugin devolver um resultado para a interface (não só imprimir)
- **Argumentos ao cadastrar** — passar valores fixos para o plugin na hora de criar o comando
- **Variáveis de ambiente** — cada plugin com seu ambiente
- **Loja de plugins** — instalar direto do app, sem baixar zip manualmente
- **Plugins WASM** — rodar plugins isolados em sandbox (mais seguro)

## Mais plataformas

- **Linux e macOS** — o esqueleto já está, falta empacotar e testar
- **Auto-atualização** — o app se atualiza sozinho quando sai versão nova
- **System tray** — ícone na bandeja do Windows com menu rápido
- **Notificação do Windows** — aviso quando um plugin termina (se for longo)

## Produtividade

- **Snippets de texto** — digitar `;email` expandir para seu e-mail completo
- **Workflows** — encadear comandos: rodar A e depois B automaticamente
- **Tarefas agendadas** — executar comandos em horários definidos
- **Macros** — gravar sequência de comandos e reproduzir com um atalho

## Backup e compartilhamento

- **Sincronização na nuvem** — `commands.json` no Dropbox/Google Drive
- **Perfis** — ter listas diferentes para casa e trabalho
- **Importar e exportar** — já existe, vai ganhar interface melhor

## Observabilidade e segurança

- **Histórico expandido** — guardar mais de 100 comandos
- **Métricas de uso** — ver quais comandos você mais usa
- **Confirmação para apps perigosos** — pedir clique duplo antes de abrir
- **Assinatura de plugins** — garantir que um plugin não foi alterado

## Ideias malucas (mas possíveis)

- **Toolbox com IA** — digitar em linguagem natural: "abre meu projeto de Python" e ele adivinha
- **Comando por voz** — falar em vez de digitar
- **Telemetria opcional** — compartilhar comandos mais usados anonimamente para sugerir plugins novos
- **Tema customizado** — você mesmo criar o tema visual
